** Internet Explorer**

  1. Click Tools
  2. Click Internet Options
  3. Click Connections
  4. Click LAN Settings
  5. Click Use a Proxy Server
  6. Click Advanced
  7. Click HTTP

**Mozilla Firefox**

  1. Click Tools
  2. Click Options
  3. Click Advanced
  4. Click Settings
  5. Click Manual Proxy Configuration

**Google Chrome**

  1. Click Options
  2. Click Settings
  3. Click Show Advanced Settings
  4. Click Change Proxy Settings
  5. Cluck LAN Settings
  6. Click Use a Proxy Server
  7. Click Advanced
  8. Click HTTP
