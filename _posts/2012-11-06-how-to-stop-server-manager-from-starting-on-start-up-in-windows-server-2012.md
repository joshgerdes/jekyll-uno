  1. **Open** **Server** **Manager**
  2. Click **Manage** _(top right)_
  3. Click **Server** **Manager** **Properties**
  4. Tick &#8220;**_Do not start Server manager automatically at logon_**&#8220;
  5. Click **Ok**
