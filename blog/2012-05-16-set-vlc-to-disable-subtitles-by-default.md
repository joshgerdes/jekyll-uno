---
title: Set VLC to disable Subtitles by default
permalink: /misc/set-vlc-to-disable-subtitles-by-default/
tags:
  - Misc
  - Windows
---
  1. Open VideoLAN (VLC)
  2. Click **Tools**
  3. Click **Preferences**
  4. Click **All**
  5. Click on **Input/Codecs**
  6. Look for **_Subtitle Track ID_**
  7. Set **ID** to “****”
  8. Press **Ok**
